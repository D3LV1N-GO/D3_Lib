I'll tell you a little about my addition to GoreBox modding. For now there are only a couple of commands, in the future I'm going to add to them and update them.

Base

Access to commands from the library is done using the print command. For example, if I want to get the time set on the PC, I should write: 

print("D3_RealDate")
The response from this command will come to the OnChatMessage function. Different commands will have different senders, but the message variable will always contain the data you wanted to receive.

Data work

To work with data dll have 2 commands
  • D3_SaveData
  • D3_GetData
These two commands do exactly the same as described earlier. They call OnChatMessage with parameters
  • sender= "D3_GetData"
  • message = YourData.
Reading occurs in the format: 
Key{YourData}
If you want to save data, use command 
print("D3_SaveData Player{Data})
You don't have to use { to save something, but it's better to always use them, it will be much more convenient to access your data in the future. The role of the separator is played by '{' I think in the future it will be possible to change it in the config. Be careful! Before you start working, you should set your path to GoreBox\D3_LibData\config.cfg. In the future, the config will have more parameters.

Real Date

The easiest command to use. There is nothing complicated here, it just reads the time from your PC and then sends everything to the OnChatMessage function. The sender variable is D3_RealDate. The message variable is the time from your PC.
